🧾 Expense Tracker (Python)

This is a simple menu-driven Expense Tracker built using Python as part of my training with Decode Labs.

:🚀 Project Overview:-

The Expense Tracker allows users to:-

-Add daily expenses
-View all recorded expenses
-Calculate total spending
-Edit or delete entries

This project focuses on data handling, logic building, and user interaction using Python.

:🛠️ Features:-
-✅ Add new expenses
-📋 View all expenses (with numbering using enumerate)
-💰 Calculate total spending
-✏️ Edit existing expenses
-🗑️ Delete expenses
-🔁 Menu-driven interface (user-friendly)

:🧠 Concepts Used:-
-Python Lists
-Functions
-Loops (while)
-Conditional Statements
-match-case (Python 3.10+)
-enumerate() and zip()
-Error Handling (try-except)
